### 1-3. 시장 기회

1. **한국 이커머스(쿠팡/네이버) 리뷰 자동 답변 전문 서비스가 거의 없음**
   - 기존 서비스는 대부분 배달앱(배민/쿠팡이츠) 중심
   - 브이리뷰가 유일한 이커머스 경쟁사이나 "리뷰 수집/마케팅" 중심 (답변 자동화가 핵심 아님)
2. 셀러들은 리뷰 답변을 해야 하는 건 알지만, 시간이 없어서 못 함
3. AI가 답변하면 퀄리티가 일정하고 빠름 → 셀러 부담 제거

## 3. 요금제 시나리오 비교 (검토 완료)

### 시나리오 A: 대표님 제안 (5만 / 19만 / 40만)

| 플랜 | 월 가격 | 내용 |
|------|---------|------|
| 스타터 | 5만원 | AI 리뷰 자동 응답 + 등록 |
| 스탠다드 | 19만원 | 위 + Q&A 응대 + 불만 알림 + 리포트 |
| 프리미엄 | 40만원 | 위 + CS 전화 대응 + 반품 안내 + 멀티채널 |

- 장점: 진입장벽 낮음 (5만원), 브이리뷰(5.5만)와 경쟁력 있음
- 단점: 스타터만 모이면 매출 낮음, 19만원으로의 점프가 3.8배로 큼

### 시나리오 B: 건당 과금 하이브리드

- 장점: 진입장벽 극극극 낮음 (500원부터), 자연스러운 업셀링
- 단점: 건당 고객은 매출 예측 어려움, 관리 복잡

### 시나리오 C: 브이리뷰 벤치마킹 (기본료 + 종량)

- 장점: 3만원 기본료로 진입장벽 매우 낮음
- 단점: 과금 구조가 복잡해 영업 시 설명 필요

## 5. 확정 요금제 (2026-03-10 v2 — CS 자동화 피봇)

### 서비스 핵심 재정의

- **기존**: 리뷰 답글 대행 중심 → 경쟁 크롬 확장(무료)과 차별화 불가
- **변경**: AI 이커머스 CS 자동화 (배송/상품문의 + 환불 1차 응대 + 전화 CS)
- **리뷰 답글**: 전 플랜 무제한 (원가 건당 1.2원, 진입 유인용 덤)
- **브랜드**: 리뷰천사 유지, 서브카피 "AI 이커머스 CS 자동화"

### 차별화 축 (변경)

| 축 | 업셀 기여도 | 비고 |
|---|---|---|
| **CS 자동응답 범위** | 최적 → 메인 축 | 셀프(배송/상품만) → 파트너(환불/교환+24시간) |
| **전화 CS 콜 수** | 높음 → 서브 축 | 전담(일10콜) → 맞춤(일30콜) |
| 연동 플랫폼 수 | 보조 | 셀프 1개 → 파트너 3개 → 전담+ 무제한 |
| 리뷰 건수 | 없음 (전 플랜 무제한) | 캡 의미 없음, 무료 확장 대비 차별화 불가 |

### 최종 요금표

| | 셀프 | 파트너 (추천) | 전담 | 맞춤 | 기업 |
|---|---|---|---|---|---|
| **월 요금** | 5.9만원 | 18만원 | 25만원 | 45만원 | 별도 협의 |
| **배송문의 자동응답** | 실시간 | 실시간 | 실시간 | 실시간 | 실시간 |
| **상품/일반 문의 자동응답** | 무제한 | 무제한 | 무제한 | 무제한 | 무제한 |
| **환불/교환 1차 응대** | — | 자동 응대 | 자동 응대 | 자동 응대 | 자동 응대 |
| **24시간 응답** | 영업시간 | 24시간 | 24시간 | 24시간 | 24시간 |
| **전화 CS** | — | — | 일 10콜 | 일 30콜 | 전담 매니저 |
| **리뷰 답글** | 무제한 | 무제한 | 무제한 | 무제한 | 무제한 |
| **불만 리뷰 알림** | 즉시 알림 | 즉시 알림 | 즉시 알림 | 즉시 알림 | 즉시 알림 |
| **연동 플랫폼** | 1개 | 3개 | 무제한 | 무제한 | 무제한 |
| **맞춤 톤 설정** | 1개 | 3개 | 무제한 | 무제한 | 무제한 |
| **리포트** | — | 주간 | 주간 | 주간+인사이트 | 맞춤 리포트 |

> 모든 플랜에 14일 무료 체험 포함 (체험 = 셀프 2주 무료)
> 셀프→파트너: 환불 응대 + 24시간이 업셀 트리거
> 파트너→전담: 전화 CS 필요성이 업셀 트리거
> 전담→맞춤: 콜 수 증가 + 인사이트 리포트가 업셀 트리거

### 티어별 역할

| 티어 | 대상 | 핵심 가치 |
|---|---|---|
| 셀프 | 1인 소규모 셀러 | "배송/상품 문의는 AI가 알아서" |
| 파트너 | 성장 셀러 (전화 불필요) | "CS 전체 24시간 자동 + 환불 응대" |
| 전담 | 전화도 맡기고 싶은 셀러 | "파트너 + 전화까지 상담원 배정" |
| 맞춤 | 멀티플랫폼 중대형 셀러 | "전화 많이 + 인사이트" |
| 기업 | 대형 셀러/브랜드 | 워크위드 콜쉐어링/도급 전환 |

### 초과 정책: 소프트 캡

- 일 콜 한도는 기준선이며 즉시 차단하지 않음
- 초과해도 당일은 계속 수신
- 주 3회 이상 초과 시 업셀 알림
- 월 5회 이상 상습 초과 → 자동 업셀 안내 메일

### 원가 구조 (Phase 1 — 셀러 ~30개사까지)

| 플랜 | 요금 | 내부 원가 | 마진 |
|---|---|---|---|
| 셀프 5.9만 | 5.9만 | AI+인프라 ~3.5만 | +2.4만 (41%) |
| 파트너 18만 | 18만 | AI 3.5만 + 상담사 ~0원 | +14.5만 (81%) |
| 전담 25만 | 25만 | AI 3.5만 + 상담사 ~0원 | +21.5만 (86%) |
| 맞춤 45만 | 45만 | AI 3.5만 + 상담사 ~0원 | +41.5만 (92%) |

- 상담사 인건비 0원 조건: 워크위드 기존 직원 유휴시간 활용
- 30개사 초과 시 상담사 채용(월 200만) → 원가 재계산 필요

### 기업 → 워크위드 본업 연결

- 대형 셀러는 전담 CS팀 필요 → 워크위드 콜쉐어링/도급 계약으로 전환
- 가격 앵커링 효과 (기업 열이 있으면 맞춤 45만이 비싸 보이지 않음)
