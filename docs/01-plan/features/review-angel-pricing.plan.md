## 1. 시장 분석

### 1-1. 국내 경쟁사

| 서비스 | 대상 | 월 가격 | 과금 방식 | 특징 |
|--------|------|---------|----------|------|
| **브이리뷰** | 이커머스 쇼핑몰 | **5.5만원/월** + 사용량 | 월정액+종량 | 리뷰 수집+AI 답변+위젯, 이커머스 전체 |
| **딜리봇** | 배달앱 | 미공개 (문의) | 월정액 추정 | 배달앱 리뷰 자동 답변 |
| **댓글몽** | 배달앱 | 미공개 (3단 요금) | 월정액 | 기본/스탠다드/스페셜, 배달앱 중심 |
| **바리뷰** | 배달앱 | 미공개 (첫달 무료) | 월정액 추정 | 배달앱 리뷰 AI 답변 |

### 1-2. 해외 경쟁사

| 서비스 | 월 가격 | 특징 |
|--------|---------|------|
| **Judge.me** | 무료 ~ $15 | 가장 저렴, Shopify 리뷰 |
| **Yotpo** | $79 ~ $169 | 리뷰 수집+마케팅 |
| **Reviews.io** | $135 ~ $235 | 리뷰 관리+SEO |
| **Birdeye** | $299 ~ $399 | 평판 관리 종합 |
| **Okendo** | $299 ~ $499 | 주문량 기반 과금 |

### 1-3. 시장 기회

1. **한국 이커머스(쿠팡/네이버) 리뷰 자동 답변 전문 서비스가 거의 없음**
   - 기존 서비스는 대부분 배달앱(배민/쿠팡이츠) 중심
   - 브이리뷰가 유일한 이커머스 경쟁사이나 "리뷰 수집/마케팅" 중심 (답변 자동화가 핵심 아님)
2. 셀러들은 리뷰 답변을 해야 하는 건 알지만, 시간이 없어서 못 함
3. AI가 답변하면 퀄리티가 일정하고 빠름 → 셀러 부담 제거

## 2. 원가 분석

| 항목 | 비용 |
|------|------|
| 리뷰 1건 처리 (입력 ~1,000토큰 + 출력 ~500토큰) | **약 1.2원** |
| 월 100건 | 약 120원 |
| 월 500건 | 약 600원 |
| 월 1,000건 | 약 1,200원 |

> **AI 비용은 거의 무시할 수 있는 수준**

### 고정 비용

| 항목 | 월 비용 (추정) |
|------|-------------|
| n8n 클라우드 | ~2만원 |
| Claude API (전체) | ~5만원 (고객 수에 따라) |
| Google Sheets (무료) | 0원 |
| 본인 시간 (운영/관리) | 인건비 |

| 항목 | 비용 |
|------|------|
| 워크위드 상담원 활용 시 | 콜쉐어링 단가 기준 (공유형) |

> 프리미엄 고객이 적을 때: 워크위드 상담원에 추가 업무로 배분 (별도 인건비 최소화)

## 3. 요금제 시나리오 비교 (검토 완료)

### 시나리오 A: 대표님 제안 (5만 / 19만 / 40만)

| 플랜 | 월 가격 | 내용 |
|------|---------|------|
| 스타터 | 5만원 | AI 리뷰 자동 응답 + 등록 |
| 스탠다드 | 19만원 | 위 + Q&A 응대 + 불만 알림 + 리포트 |
| 프리미엄 | 40만원 | 위 + CS 전화 대응 + 반품 안내 + 멀티채널 |

- 장점: 진입장벽 낮음 (5만원), 브이리뷰(5.5만)와 경쟁력 있음
- 단점: 스타터만 모이면 매출 낮음, 19만원으로의 점프가 3.8배로 큼

### 시나리오 B: 건당 과금 하이브리드

- 장점: 진입장벽 극극극 낮음 (500원부터), 자연스러운 업셀링
- 단점: 건당 고객은 매출 예측 어려움, 관리 복잡

### 시나리오 C: 브이리뷰 벤치마킹 (기본료 + 종량)

- 장점: 3만원 기본료로 진입장벽 매우 낮음
- 단점: 과금 구조가 복잡해 영업 시 설명 필요

## 4. 시나리오별 종합 비교

| 기준 | A (5/19/40) | B (건당+월정액) | C (기본료+종량) |
|------|-------------|---------------|--------------|
| **진입장벽** | 낮음 (5만) | **최저 (500원)** | 매우 낮음 (3만) |
| **업셀링** | 점프 큼 (5→19) | **자연스러움** | 자연스러움 |
| **매출 예측** | **쉬움** | 어려움 | 중간 |
| **영업 설명** | **가장 간단** | 중간 | 복잡 |
| **브이리뷰 대비** | 동등 (5만 vs 5.5만) | 저렴 | 저렴 |
| **수익성** | 좋음 | 변동적 | 좋음 |
| **셀러 심리** | "5만원이면 해볼까" | **"500원이면 당장"** | "3만원이면 해볼까" |

## 5. 확정 요금제 (2026-03-09)

### 차별화 축 분석

| 축 | 원가 연동 | 차별화 적합도 |
|---|---|---|
| **CS 전화 콜 수** | 직결 (상담사 인건비 = 통화시간 비례) | **최적 → 메인 축** |
| **연동 플랫폼 수** | 거의 없음 (스크래퍼 일회성) | 원가 근거 약하지만 업셀 명분 강함 → **보조 축** |
| 리뷰 건수 | 미미 (AI 100원/건) | 약함 |
| 배송추적 건수 | 중간 (SweetTracker 종량) | 보조 가능 |
| 상품 수 / 응답 속도 SLA | 없음 | 부적합 |

### 최종 요금표

| | Lite | S (추천) | L | 기업 맞춤 |
|---|---|---|---|---|
| **월 요금** | 5.9만원 | 18만원 | 42만원 | 별도 협의 |
| **AI 리뷰 자동답변** | 월 300건 | 월 500건 | 무제한 | 무제한 |
| **연동 플랫폼** | 3개 | 5개 | 무제한 | 무제한 |
| **배송추적 자동응답** | O | O | O | O |
| **Q&A/문의 자동응답** | — | O | O | O |
| **불만 리뷰 알림** | O | O | O | O |
| **전화 상담** | — | 일 10콜 | 일 30콜 | 전담 배정 |
| **맞춤 톤 설정** | 1개 | 3개 | 무제한 | 무제한 |
| **리포트** | — | 주간 | 주간+인사이트 | 맞춤 리포트 |
| **전담 매니저** | — | — | — | O |

> 모든 플랜에 14일 무료 체험 포함 (체험 = Lite 2주 무료)
> Lite AI 리뷰 300건 초과 시 → S 업셀 유도
> S 이상 필요 시 → 기업 맞춤 상담으로 자연 전환

### 초과 정책: 소프트 캡

- 일 콜 한도는 기준선이며 즉시 차단하지 않음
- 초과해도 당일은 계속 수신
- 주 3회 이상 초과 시 업셀 알림
- 월 5회 이상 상습 초과 → 자동 업셀 안내 메일

### 원가 구조 (Phase 1 — 셀러 ~30개사까지)

| 플랜 | 요금 | 내부 원가 | 마진 |
|---|---|---|---|
| Lite 5.9만 | 5.9만 | AI+인프라 ~3.5만 | +2.4만 (41%) |
| S 18만 | 18만 | AI 3.5만 + 상담사 ~0원 | +14.5만 (81%) |
| L 42만 | 42만 | AI 3.5만 + 상담사 ~0원 | +38.5만 (92%) |

- 상담사 인건비 0원 조건: 워크위드 기존 직원 유휴시간 활용
- 30개사 초과 시 상담사 채용(월 200만) → 원가 재계산 필요

### 기업 맞춤 → 워크위드 본업 연결

- 대형 셀러는 전담 CS팀 필요 → 워크위드 콜쉐어링/도급 계약으로 전환
- 가격 앵커링 효과 (기업 문의 열이 있으면 L이 비싸 보이지 않음)
