# [Plan] 견적서 업종별 문제-해결책 패키징

> 작성일: 2026-02-23
> 상태: **Draft**

---

## 1. 개요

### 1.1 목적

견적 시뮬레이터에서 업종을 선택하면, 해당 업종의 **일반적인 고민(Pain Point)**과 **워크위드의 솔루션**을 견적서에 자동 포함하여, 단순 가격 제시가 아닌 **업종 맞춤 패키징**으로 영업 경쟁력을 강화한다.

### 1.2 배경

- 현재 견적서에는 `getServiceDescription(type)`으로 **견적 유형별** 서비스 설명만 존재
- 같은 "정액제 도급"이라도 세차업과 제조업의 고민은 완전히 다름
- 업종별 고민 → 솔루션 매핑을 견적서에 넣으면 "우리를 왜 써야 하는지"가 명확해짐
- 기존 업종 분류도 재정리 필요 (사무/CS 분리, 이커머스 추가)

### 1.3 관련 파일

- `docs/quotation-tool.html` — 견적 시뮬레이터 (수정 대상)
- 기존 함수: `getServiceDescription(type)`, `onIndustryChange()`

### 2.2 제외 (Out of Scope)

- 리뷰천사(Review Angel) 관련 내용은 견적서에 포함하지 않음
- 구체적 단가/과금 정보는 패키징에 넣지 않음 (단가 협상 시 별도)
- 견적 유형별 서비스 설명(`getServiceDescription`)은 기존 유지

### 3.4 업종별 패키징 내용 (4항목씩)

각 업종별로 **고민 → 솔루션** 4쌍을 작성한다.

**[세차/주차]**
| 업종 고민 | 워크위드 솔루션 |
|----------|--------------|
| 잦은 퇴사로 인력 공백 발생 | 자체 인력풀 보유, 당일 대체인력 투입 보장 |
| 서비스 품질 편차가 큼 | 입사 전 서비스 마인드 교육 + 정기 CS교육 실시 |
| 직접 고용 시 4대보험·노무 부담 | 당사가 4대보험·급여·노무관리 일체 처리 |
| 성수기/비수기 인원 조절 어려움 | 탄력배치 운영으로 비수기 비용 절감 |

**[물류/택배]**
| 업종 고민 | 워크위드 솔루션 |
|----------|--------------|
| 야간·새벽 근무자 확보 어려움 | 야간 전문 인력 보유, 수당 포함 단가 제시 |
| 물량 변동에 따른 인력 유동성 필요 | 일·주 단위 증감 가능한 탄력 운영 체계 |
| 산재사고 리스크 높음 | 안전교육 의무 실시 + 산재보험 완비 |
| 인력 관리에 시간·비용 과다 소모 | 현장관리자 상주 배치, 근태·품질 일괄 관리 |

**[제조 라인]**
| 업종 고민 | 워크위드 솔루션 |
|----------|--------------|
| 불법파견 판정 리스크 | 독립 지휘체계·공정분리로 적법도급 운영 (8년 무사고) |
| 생산라인 중단 시 매출 손실 | 결원 발생 당일 대체인력 투입, 라인 중단 최소화 |
| 신규 인력 숙련도 낮음 | 자체 직무교육 프로그램으로 투입 전 사전 교육 |
| 근로감독 대응 어려움 | 적법운영안내서 제공 + 근로감독 공동 대응 |

**[사무/행정]**
| 업종 고민 | 워크위드 솔루션 |
|----------|--------------|
| 채용 후 조기 퇴사 반복 | 적합성 검증 후 배치 + 조기퇴사 시 즉시 교체 |
| 전문 인력 채용 비용 부담 | 도급 전환으로 채용·교육·관리 비용 일괄 절감 |
| 업무 표준화·인수인계 어려움 | 매뉴얼 기반 업무 정립 + 체계적 인수인계 지원 |
| 단기 프로젝트 인력 필요 | 기간 맞춤 인력 배치, 계약기간 자유 설정 |

**[CS/콜센터]**
| 업종 고민 | 워크위드 솔루션 |
|----------|--------------|
| 자체 콜센터 운영비 과다 (CTI·녹취 등) | ASP 시스템 일체 포함, 별도 인프라 비용 없음 |
| 상담 품질 관리가 안 됨 | QA 전담 + AI 답변 시스템으로 정확도 향상 |
| 상담원 채용·교육 부담 | 전문 상담사 직접 채용·교육 후 배치 |
| 상담 데이터 활용 못함 | 일간/주간 VOC 리포트 + 녹취 DB 제공 |

**[이커머스]**
| 업종 고민 | 워크위드 솔루션 |
|----------|--------------|
| 고객 문의·반품·교환 응대에 시간 소모 | 전문 상담사가 CS 전반 대행 처리 |
| CS 매뉴얼이 없어 응대 품질 들쭉날쭉 | 업체별 맞춤 매뉴얼 구축 + AI 답변 시스템 연동 |
| CS 때문에 마케팅·상품에 집중 못함 | CS 통합 대행으로 핵심 업무 집중 환경 제공 |
| 1인·소규모 셀러는 전담 상담사 부담 | 쉐어링 방식으로 소규모도 전문 CS 이용 가능 |

**[경비/시설관리]**
| 업종 고민 | 워크위드 솔루션 |
|----------|--------------|
| 24시간 교대근무 인력 확보 난항 | 격일근무·3교대 전문 인력 보유 |
| 고령 인력 관리 어려움 | 체력·적합성 평가 후 배치 + 정기 건강관리 |
| 야간 근무 중 사고 리스크 | 안전교육 정기 실시 + 산재보험 완비 |
| 경비 인력 서비스 마인드 부족 | 입주민 응대 교육 + 서비스 마인드 정기 교육 |

**[호텔/숙박]**
| 업종 고민 | 워크위드 솔루션 |
|----------|--------------|
| 서비스 품질이 곧 매출에 직결 | 호텔 서비스 전문교육 이수 인력 배치 |
| 객실 청소·프론트 인력 잦은 이탈 | 자체 인력풀 활용 당일 충원 보장 |
| 성수기/비수기 인력 수급 불균형 | 시즌별 탄력배치 + 비수기 비용 최적화 |
| 외국어 가능 인력 부족 | 외국어 가능 인력 우선 매칭 서비스 |

**[식품 제조]**
| 업종 고민 | 워크위드 솔루션 |
|----------|--------------|
| 위생관리 기준 미충족 시 영업정지 | 위생교육 의무화 + HACCP 기준 준수 관리 |
| 생산량 변동에 따른 인력 유동성 필요 | 일·주 단위 인원 증감 가능 탄력 운영 |
| 불법파견 판정 리스크 | 독립 공정분리 + 적법운영안내서 제출 |
| 제조 경험 인력 확보 어려움 | 식품제조 경험자 인력풀 보유 + 사전 교육 |

**[기타]**
| 업종 고민 | 워크위드 솔루션 |
|----------|--------------|
| 인력 채용·관리에 시간 소모 | 채용부터 관리까지 원스톱 아웃소싱 |
| 인건비 외 부수 비용(4대보험 등) 부담 | 복리후생비·4대보험 포함 단가, 추가비용 없음 |
| 갑작스러운 결원 대응 어려움 | 자체 인력풀 활용 당일 대체 투입 보장 |
| 노무 리스크(근로감독 등) 부담 | 적법 운영 + 노무사 자문 지원 |

### 4.2 getIndustryPackage(industry) 함수

- 입력: 업종 select의 value
- 출력: HTML 문자열 (고민-솔루션 테이블)
- 업종 미선택 시 빈 문자열 반환
- 기존 `getServiceDescription(type)`과 별도 함수로 구현
- 견적서에서는 **서비스 설명 아래**에 배치

### 4.3 견적서 삽입 위치

```
견적 금액 테이블
  ↓
★ 서비스 안내 (getServiceDescription - 견적 유형별)
  ↓
★ 업종별 솔루션 (getIndustryPackage - 업종별)  ← 신규
  ↓
기타사항 / 특기사항
```

### 4.4 하위 호환

- `loadSaved()`에서 기존 "사무/CS" → "사무/행정"으로 매핑
- `loadFromHistory()`에서도 동일 매핑 적용
