# 워크위드 영업응대 온보딩 가이드

> 신입 영업 담당자를 위한 전화응대 → 서비스 분류 → 팔로업 실전 가이드

---

## 1. 전화 받으면 이것부터 (10가지 체크리스트)

"안녕하세요, 워크위드 [이름]입니다." 인사 후 아래 항목을 하나씩 물어보세요.

| # | 항목 | 질문 예시 |
|---|------|----------|
| 1 | 업체명 | "어떤 회사에서 연락 주셨나요?" |
| 2 | 담당자 이름 / 연락처 / 이메일 | "담당자분 성함과 연락처, 이메일 부탁드립니다" |
| 3 | 업종 | "어떤 업종이신가요?" (제조, 서비스, IT, 병원 등) |
| 4 | 필요 인원 수 | "몇 분 정도 필요하세요?" |
| 5 | 업무 내용 | "어떤 업무를 하실 분이 필요하세요?" |
| 6 | 근무 시간 | "근무 시간은 어떻게 되나요?" (9-18, 교대 등) |
| 7 | 기간 (상시 / 단기) | "기간은 상시인가요, 단기인가요?" |
| 8 | 근무 지역 | "근무 지역이 어디인가요?" |
| 9 | 도입 시기 | "언제부터 필요하세요?" |
| 10 | 유입 경로 | "저희를 어디서 보고 연락 주셨나요?" |

---

## 2. 마무리 멘트

### 서비스 유형이 파악됐을 때
> 말씀하신 내용 확인했습니다. 견적서 준비해서 내일 중으로 메일로 보내드리겠습니다. 추가로 궁금하신 점 있으시면 편하게 연락 주세요.

### 아직 유형이 안 잡힐 때
> 말씀하신 내용 정리해서, 가장 적합한 방식으로 견적을 준비하겠습니다. 내일 중으로 연락드리겠습니다.

### 고객이 바빠서 빨리 끊으려 할 때
> 간단한 내용만 여쭤보고, 상세 견적은 메일로 보내드리겠습니다. 업체명, 필요 인원, 업무, 연락처만 알려주시면 됩니다.

## 3. 절대 하면 안 되는 말

- ~~"저희는 그런 업무 안 합니다"~~
- ~~"그건 힘들 것 같습니다"~~
- ~~"조건이 안 맞아서요"~~
- ~~"예전에 해봤는데 안 됐어요"~~

> **이 말 하는 순간 고객은 끊고 다른 업체에 전화합니다.**
> 모든 전화 한 통에 광고비가 들어가 있습니다.

**혼자 거절 가능한 유일한 경우:** 불법 업무(도박, 성인업종), 사업자 없는 개인 1일 요청

**그 외 전부:** "확인 후 안내드리겠습니다" → 관리자에게 보고

## 4. 워크위드 서비스 분류 (6가지)

우리 서비스는 6가지입니다. 콜센터는 3가지 하위 유형이 있습니다.

### 4-1. 도급(상시)

- **한마디로:** 채용+교육+관리+급여 전부 우리가
- **적합:** 제조/물류, 10명+, 6개월+
- **핵심:** "결과물 납품" — 현장관리자(워크위드 소속) 상주, 고객사는 작업 지시를 현장관리자에게만 함
- 4대보험, 급여, 교육 모두 워크위드 책임

> **응대 멘트:** 저희가 인력 채용부터 교육, 현장 투입, 관리까지 일괄로 해드리는 도급 방식이 적합합니다. 현장관리자가 상주하며 인원을 관리하고, 고객님께서는 결과물만 받으시면 됩니다. 상세 견적 보내드릴까요?

### 4-2. 도급(단기)

- **"아르바이트"는 별도 서비스가 아님** — 누가 급여를 주고 소득신고를 하느냐에 따라 **단기 도급** 또는 **알선**으로 나뉨
- **기간 제한 없음** — 1일~수개월, 규모/성격으로 상시 도급과 구분

#### A. 단기 도급

- **급여: 워크위드가 지급** / **소득신고: 워크위드가 신고**
- 워크위드 소속으로 투입 (4대보험 or 일용직 신고)
- 관리도 워크위드 (현장관리자 배치 또는 지시)
- 고객사는 결과물만 받음

#### B. 알선

- **급여: 고객사가 직접 지급** / **소득신고: 고객사가 직접 신고**
- 사람만 소개해주고, 고용은 고객사 ↔ 근로자 직접 체결
- 워크위드는 소개 수수료만 받음

> **핵심 질문:** 인력 급여를 저희가 지급하고 관리까지 해드리는 방식과, 인력만 소개해드리고 급여는 고객사에서 직접 지급하시는 방식이 있는데요, 어떤 방식을 원하시나요?

### 4-3. 파견

- **한마디로:** 사람 보내고 급여 지급, 관리는 고객사
- **적합:** 사무/전문직, 권장 5명+, 3개월+
- **핵심:** "사람을 보내드림" — 고객사 지휘 아래 근무, 교육/관리는 고객사 책임
- 4대보험, 급여는 워크위드가 지급

> **응대 멘트:** 저희 소속 인력을 고객사에 파견해드리고, 급여와 보험은 저희가 처리합니다. 현장 관리는 고객사에서 직접 하시는 구조입니다. 견적 보내드릴까요?

> **⚠️ 파견법 주의:** 생산직(제조/포장)은 파견 불가 → 도급으로 안내. "도급 형태로 계약하시는 게 법적 리스크 없습니다"

### 4-4. 채용대행
- **한마디로:** 인재 매칭, 채용 확정 시 수수료
- **적합:** 전 업종, 1~5명, 1회성
- **핵심:** "사람 찾아드림" — 구인공고, 지원자 스크리닝, 면접 연결까지 대행
- 채용 확정 시에만 수수료 발생 (리스크 낮음)

> **응대 멘트:** 저희가 적합한 인재를 찾아서 면접까지 연결해드리고, 채용 확정 시에만 수수료가 발생하는 채용대행 방식입니다. 부담 없이 진행 가능합니다. 견적 보내드릴까요?

### 4-6. 콜센터 (3가지 하위 유형)

콜량과 운영 방식에 따라 3가지 유형:

| 유형 | 콜량 | 월 비용 | 특징 |
|------|------|---------|------|
| 콜센터 대행 (전담) | 200콜+ | 300만원~ | 고객사 전용 상담팀 |
| CS대행 (반전담) | 100콜 내외 | 150만원~ | CS 통째 위탁 운영 |
| CS 쉐어링 (공유) | 50콜 이하 | 50~80만원 | 상담원 공유, 콜당 과금 |

> **응대 멘트:** 저희 콜센터에서 CS 업무를 대행해드립니다. 월 콜량이 대략 어느 정도 되시나요? 콜량에 따라 전담, 공유 등 비용 효율적인 방식을 안내해드리겠습니다.

### 4-5. 알선
- **한마디로:** 구인·구직 연결, 소개 수수료
- **적합:** 1명~소규모, 기간 무관, 특정 직무/기술 불필요
- **핵심:** "사람 연결해드림" — 고용계약은 고객사 ↔ 근로자 직접 체결
- 관리/급여/교육 모두 고객사 책임, 워크위드는 소개 수수료만 (1회성)

> **응대 멘트:** 저희가 조건에 맞는 인력을 찾아서 소개해드리고, 고용은 고객사에서 직접 하시는 알선 방식입니다. 소개 수수료만 발생하고, 부담 없이 진행 가능합니다.

## 5. 헷갈리는 경계 구분

비슷해 보이지만 다릅니다. 핵심 차이를 기억하세요.

### 알선 vs 채용대행

| | 알선 | 채용대행 |
|--|------|---------|
| 방식 | 단순 연결 (소개만) | 스크리닝+면접까지 |
| 요건 | 특별한 스킬 불필요 | 특정 스킬/경력 필요 |
| 수수료 | 저렴 | 높음 |
| 요청 패턴 | "아무나 괜찮은" 요청 | "이런 사람 찾아줘" 요청 |

> **판단 기준:** "아무나 괜찮아요" → 알선 / "경력 3년 이상 OO 가능한 분" → 채용대행

### 단기 도급 vs 알선 ("아르바이트 필요해요" 라고 하면?)

| | 단기 도급 | 알선 |
| -- | -------- | ---- |
| 급여 지급 | **워크위드** | **고객사** |
| 소득신고 | **워크위드** | **고객사** |
| 관리 | 워크위드 | 고객사 |
| 고용관계 | 워크위드 소속 | 고객사 ↔ 근로자 직접 |
| 비용 | 일당/주급 (도급비) | 소개 수수료 |

> **판단 기준:** "급여/관리 다 맡길게요" → 단기 도급 / "사람만 보내줘요, 급여는 우리가" → 알선

### 도급 vs 파견

| | 도급 | 파견 |
|--|------|------|
| 관리 | 우리 (현장관리자 상주) | 고객사가 직접 관리 |
| 결과물 | 고객사는 결과물만 받음 | 고객사 지휘 아래 근무 |
| 지시 | 근로자에게 직접 지시 X | 근로자에게 직접 지시 O |
| 생산직 | 가능 | 불가 (파견법) |

> **판단 기준:** "관리까지 해주세요" → 도급 / "사람만 보내주세요" → 파견 / 생산직이면 무조건 → 도급

### CS대행 vs CS쉐어링 vs 콜센터 전담

콜 수로 구분하면 간단합니다:

- 월 50콜 이하 → **CS쉐어링** (상담원 공유, 가성비)
- 월 100콜 내외 → **CS대행** (반전담, CS 통째 위탁)
- 월 200콜 이상 → **콜센터 전담** (전용 상담팀 배치)
- 콜 수 모르면? → "대략적인 월 문의량" 물어보세요

### 경계가 애매하면?

- 혼자 판단하지 말고 → **관리자에게 보고 후 확인**
- 고객에게는: "가장 적합한 방식 확인해서 안내드리겠습니다"
- 시트에 고객 요청 내용을 **구체적으로** 적어두기

## 6. 서비스 전환 (안 맞으면 바꿔서 제안)

**"안 됩니다"로 끝내지 말고, "다른 방법이 있습니다"로 이어가세요.**

### 상황 1: 채용대행인데 "교육도 해달라"
- ~~"채용대행은 구인, 이력서, 면접까지만 가능합니다"~~
> 교육까지 포함하시면 도급 형태가 적합합니다. 저희가 인력 채용부터 교육, 현장관리까지 일괄로 해드리는 방식인데요, 견적 보내드릴까요?

### 상황 2: 파견인데 1~2명 소규모
- ~~"소규모는 파견이 어렵습니다"~~
> 소규모의 경우 채용대행이 더 효율적입니다. 적합한 분을 찾아서 면접까지 연결해드리고, 채용 확정 시에만 수수료가 발생해서 부담이 적으세요. 견적서 보내드릴까요?

### 상황 3: 파견인데 생산직
- ~~(그냥 파견으로 진행 → 파견법 위반 위험)~~
> 생산직의 경우 파견법상 도급 형태로 계약하시는 게 안전합니다. 실질적인 운영은 동일하고, 법적 리스크 없이 진행할 수 있어요. 도급 기준으로 견적 보내드릴까요?

### 상황 4: 콜센터 전담인데 콜량 적음 (월 100건 미만)
- ~~"콜량이 적어서 전담은 어렵습니다"~~
> 콜량 기준으로 보면 쉐어(공유) 형태가 비용 효율적입니다. 전담 대비 비용이 절반 이하이고, 품질은 동일하게 관리됩니다. 쉐어와 전담 비교 견적 함께 보내드릴까요?

### 상황 5: "단기로 사람 필요해요"

- ~~"기간이 너무 짧아서 어렵습니다"~~
> 단기 인력도 가능합니다. 저희가 급여 지급과 관리까지 해드리는 방식과, 인력만 소개해드리는 방식이 있는데요, 어떤 쪽이 편하세요?
> (급여/관리 맡기면 → 단기 도급 / 사람만 소개 → 알선)

### 상황 6: CS쉐어링인데 "우리 회사 전용으로 해달라"
- ~~"쉐어링은 공유 방식이라 전담은 안 됩니다"~~
> 전용 상담원 배치를 원하시면 콜센터 전담 방식이 적합합니다. 고객사 전용 팀으로 운영되고, 전문적인 대응이 가능합니다. 쉐어링과 전담 비교 견적 함께 보내드릴까요?

### 상황 7: 도급 요청인데 3명 소규모
- ~~"3명은 도급이 어렵습니다"~~
> 소규모의 경우 채용대행이나 파견이 더 효율적입니다. 채용대행은 채용 확정 시에만 수수료가 발생하고, 파견은 저희가 급여를 지급하는 구조입니다. 두 가지 비교 견적 보내드릴까요?

### 전환이 뭔지 모르겠을 때
> 고객님 요청에 맞는 최적의 방식을 확인해서 내일 중 안내드리겠습니다.
- 시트에 기록 → 관리자에게 보고 → 관리자가 결정

## 7. 고객이 자주 하는 질문 (FAQ)

### "단가가 어떻게 되나요?"
> 인원수, 근무 조건에 따라 달라지는데요, 말씀하신 내용 기준으로 견적서 보내드리겠습니다. 메일 주소 알려주시겠어요?
- 전화에서 즉석 단가를 말하지 마세요. 견적서로 보내야 비교 검토가 가능하고, 기록도 남습니다.

### "수수료/마진이 얼마나 되나요?"
> 수수료율은 인원 규모와 계약 조건에 따라 다르고요, 견적서에 상세하게 안내드리겠습니다.
- 수수료율을 전화에서 구체적으로 말하지 마세요.

### "급하게 내일부터 필요해요"
> 내일 투입은 준비 기간이 필요해서 어렵지만, 빠른 시일 내 인력 구성해서 투입 가능합니다. 가장 빠르게 가능한 일정 확인 후 바로 안내드리겠습니다.
- "안 됩니다"가 아니라 "언제 가능한지"를 안내합니다.

### "다른 데서도 견적 받고 있어요"
> 네, 비교 검토하시는 거 당연합니다. 저희 견적 함께 검토해보시고, 궁금한 점 있으시면 편하게 연락 주세요.
- 위축되지 마세요. 견적 보내고 3일 후 반드시 팔로업 하세요.

### "생각해보고 연락할게요"
> 네, 충분히 검토하시고 편하실 때 연락 주세요. 혹시 다음 주 중으로 제가 한번 확인 연락드려도 될까요?
- "연락할게요"는 90%가 안 합니다. 우리가 먼저 팔로업해야 합니다.

### "4대보험은 어떻게 되나요?"
> 도급이나 파견의 경우 저희 소속으로 4대보험 가입되고, 채용대행의 경우 고객사에서 직접 가입하시는 구조입니다. 견적서에 보험료 포함/미포함 구분해서 보내드리겠습니다.

### 고객이 "됐다"고 끊으려 할 때
- ~~(그냥 끊기)~~
> 혹시 다른 방식으로도 진행 가능한 부분이 있는데, 간단히 안내드려도 될까요?
- "안 돼서 끊는 게 아니라, 다른 방법이 있다"는 걸 알려주세요.

### "금액 좀 깎아주세요"
> 확인 후 답변드리겠습니다.
- 절대 혼자 네고하지 않기. 관리자에게 보고 후 결정.

## 8. 견적 발송 후 팔로업

**견적 보내고 끝이 아닙니다. 팔로업이 계약을 만듭니다.**

| 시점 | 액션 | 멘트 |
|------|------|------|
| 발송 당일 | 발송 확인 문자 | "견적서 메일로 보내드렸습니다. 확인 부탁드립니다. - 워크위드" |
| 발송 +3일 | 1차 팔로업 전화 | "안녕하세요, 워크위드입니다. 보내드린 견적 검토해보셨나요? 궁금한 점 있으시면 안내드리겠습니다." |
| 발송 +7일 | 2차 팔로업 전화 | "안녕하세요, 워크위드입니다. 추가로 궁금하신 점이나 조건 조정이 필요하시면 편하게 말씀해 주세요." |
| 발송 +14일 | 최종 팔로업 → 응답 없으면 "보류" | "안녕하세요, 워크위드입니다. 혹시 인력 관련 진행 상황이 어떻게 되셨는지 확인 연락드렸습니다." |

### 팔로업 중 고객 반응별 응대

| 반응 | 멘트 | 행동 |
|------|------|------|
| "검토 중이다" | "네, 편하실 때 연락 주세요. 다음 주에 한번 더 확인드려도 될까요?" | 시트에 팔로업 일정 기록 |
| "금액 조정 가능해요?" | "확인 후 답변드리겠습니다." | 관리자에게 보고. 절대 혼자 네고 X |
| "다른 데로 정했어요" | "네, 알겠습니다. 추후 인력 필요하시면 편하게 연락 주세요." | 시트에 "타사 선정" 기록 후 종결 |
| 전화 안 받음 | "워크위드입니다. 보내드린 견적 관련 확인 연락드렸습니다. 편하실 때 연락 부탁드립니다." | 다음 날 재시도 → 문자 → 3회 시도 후 보류 |

## 9. 시트 기록 방법

```
3/8 초기컨택 - 식품제조업, 포장 인력 20명 필요
    3월 중순 투입 희망, 통근버스O, 외국인OK

3/12 1차 팔로업 - 검토 중, 다음 주 회신 예정
[다음 액션] 3/19 2차 팔로업 전화
```

### 나쁜 기록 예시
```
3/8 초기컨택
[다음 액션] (비어있음)
→ 이러면 다음에 아무도 이 건을 이어갈 수 없음
```

> **"다음 액션" 칸이 비어있으면 안 됩니다**
> - "3/12까지 견적서 발송"
> - "3/15에 팔로업 전화"
> - "관리자 판단 대기 중"

## 11. 빠른 판단 가이드

### 인원 수로 판단

| 인원 | 판단 |
|------|------|
| 10명 이상 | 무조건 진행 (대형 건). 도급 or 파견 |
| 5~9명 | 진행 가능. 도급/파견 판단 |
| 3~4명 | 장기(6개월+)면 진행, 단기면 관리자 확인 |
| 1~2명 | 채용대행 안내. 거절하면 관리자 연결 |

### 업종/상황으로 판단

| 업종/상황 | 추천 서비스 |
|----------|-----------|
| 제조/생산/포장/물류 | → 도급 (관리 포함, 파견법상 파견 불가) |
| 사무/전문직 | → 파견 (관리는 고객사) |
| 전 업종 소규모(1~5명) | → 채용대행 (리스크 낮음) |
| CS 대규모(200콜+) | → 콜센터 대행 전담 |
| CS 중규모(100~200콜) | → CS대행 반전담 |
| CS 소규모(50콜 이하) | → CS 쉐어링 |
| "급여/관리 맡길게요" 단기 | → 단기 도급 (워크위드가 급여+소득신고) |
| "사람만 보내줘요" 소개 | → 알선 (고객사가 급여+소득신고) |
| 모르겠다 | → 관리자에게 넘기기 |
